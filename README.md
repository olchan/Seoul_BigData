# 2024 서울시 빅데이터 캠퍼스 공모전 우수상

## 🧩 Project

: 다양한 도시 데이터를 활용해 서울시 행정동 간 연결성을 극대화하는 선형공원 입지를 클러스터링 + 유전 알고리즘으로 도출한 프로젝트


🚨 Problem Definition
- 서울은 도로·철도로 인해 행정동 간 물리적 단절 존재
- 공원이 산발적으로 분포되어 연결성이 부족
- 1인당 공원 면적 부족 및 녹지 불균형 심화
- 특정 상권으로의 경제 집중 현상

👉 단순 녹지 확장이 아닌 “연결을 만드는 공원” 설계 문제로 재정의

## 🛠
- Python (Pandas, Scikit-learn)
- K-Means Clustering
- Genetic Algorithm
- GIS (QGIS)
- Optimization Modeling
